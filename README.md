College Enquiry Chatbot

The College Enquiry Chatbot is an AI-based application designed to help students and parents get quick answers to common college-related queries. It provides instant responses about courses, admissions, fees, hostel facilities, college timings, and placement details without the need for manual assistance.

This chatbot uses basic Natural Language Processing (NLP) to understand user questions and respond in simple, clear English. It reduces the workload of college staff and ensures users receive information anytime, anywhere.

The project is beginner-friendly and ideal for students learning Artificial Intelligence and Machine Learning. It can be easily customized with different college details and expanded with advanced AI features in the future.
